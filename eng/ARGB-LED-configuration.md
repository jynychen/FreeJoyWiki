## Connecting WS2812, PL9823 ARGB LEDs:

Addressable LEDs should be connected to pin A10 via a 1K pull-up resistor to 5V.

![LARGB1](../images/LARGB1.jpg)

## Settings:

The LED/PWM tab contains LED settings. Connect the LEDs as shown in the image above, select the number in the LEDs count and the desired effect. You can select LEDs in the list with the mouse and configure multiple LEDs at once.

Effects:
1. Static Color - the LED glows in one specified color.
2. Rainbow - the LEDs shimmer with rainbow colors at a configurable interval and brightness.
3. Flow - the LEDs transfer their color to the next LED at a configurable interval.
4. SimHub - control LEDs via the [SimHub] app (Connecting to SimHub.md).

![LARGB2](../images/LARGB2.jpg)

## Buttons:

You can right-click on an LED and select which logical button will turn it on or off.
